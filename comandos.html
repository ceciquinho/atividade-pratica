Comandos Básicos do Ubuntu

Páginas Estáticas Ubuntu

Principais Comandos

  • ls: Lista os arquivos e diretórios no diretório atual.
  • cd: Navega entre diretórios (ex: cd /home).
  • sudo: Executa comandos com privilégios de administrador.
  • apt-get: Gerencia pacotes (ex: sudo apt-get update para atualizar os repositórios).
  • mkdir: Cria um novo diretório (ex: mkdir nome_diretorio).
